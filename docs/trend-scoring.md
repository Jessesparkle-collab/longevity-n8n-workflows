# X 热点评分逻辑

## 目标

不是寻找绝对互动量最高的帖子，而是寻找在短时间内引发真实讨论、值得垂直媒体参与的帖子。

## 候选获取

- 搜索窗口：最近 8 小时
- 主题：长寿、抗衰、健康寿命、代表性干预与生物标志物
- 去除重复 URL 或重复正文
- 排除推广垃圾和低质量内容

## 基础门槛

```text
reply_count >= 5
like_count >= 20
reply_count / like_count >= 0.03
```

此外，工作流排除一组已经长期跟踪的头部 KOL，让榜单更容易发现中小账号出现的异常讨论。

## 评分公式

```text
engagement_score = reply_count * 5
                 + repost_count * 2
                 + like_count * 0.5
```

回复权重最高，因为它比轻量点赞更能代表讨论意愿；转发代表传播；点赞用于提供基础认可度。

示例：

```text
14 replies, 0 reposts, 24 likes
= 14 * 5 + 0 * 2 + 24 * 0.5
= 82
```

## 下游处理

- Top 5：翻译、长寿领域切入点、回复建议和评论氛围
- Top 2：进入文章生成与基础质量检查
- 输出：飞书运营卡片，并可配置为 WordPress 草稿或发布

## 已知边界

当前版本没有使用粉丝数归一化、浏览量归一化或连续时间衰减。时间约束来自“最近 8 小时”的检索窗口。公开说明必须保持这一口径。
