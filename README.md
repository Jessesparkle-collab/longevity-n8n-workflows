# Longevity n8n Workflows

Redacted n8n workflow templates for automated longevity science monitoring and WordPress publishing.

## Workflows

- `workflows/cns-automation.redacted.json`  
  CNS automation workflow covering Cell, Nature, and Science pipelines with AI relevance filtering, AI content generation, and WordPress publishing.

## Security

This repository contains redacted workflow templates only.

Removed or replaced before publishing:

- n8n credential IDs and names
- WordPress site credentials and private endpoint details
- Google Sheet IDs and cached URLs
- n8n instance metadata and webhook IDs
- literal API keys, tokens, and secrets

To run these workflows in n8n, import the JSON file and reconnect your own credentials inside n8n.

## Notes

Public journal URLs and public API endpoints are preserved so the workflow logic remains understandable.
